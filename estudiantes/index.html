Talleres

Talleres para el semestre 2017-I.

Taller Horario y aula Profesor/a Objetivos
Taller de Robótica Verificar los anuncios hechos por el Prof. Dennis Barrios con respecto a inscripciones LARVIC y entrar en contacto con él mismo Dennis Barrios (dennisbarrios@gmail.com)

Taller de HCI Miercoles de 10:00 a 12:15, Laboratorio G, Campus San Lázaro. Manuel Eduardo Loaiza Fernandez

El taller busca analizar e investigar las nuevas tecnologías de interacción 3D usando dispositivos para realidad virtual y aumentada. Combinar las tecnologías y poder crear nuevas formas de complementar la sensación de inmersión de un usuario al usar los diferentes dispositivos de interacción.

Recomendable: Saber programar en algun lenguaje como : C , C++, C# o Python.

Taller de Programación Competitiva (ACM) Martes de 13:00 a 15:15 h. Laboratorio C, Campus San Lázaro. Joaquin Garmendia (joaquin.garmendia@ucsp.edu.pe)

El taller busca enseñar como resolver problemas algorítmicos similares a los que se usan en concursos/olimpiadas de programación y motivar el estudio de algoritmos, programación y matemática. Se estudiaran las técnicas algorítmicas para poder resolver estos problemas como programación dinámica, backtracking, estructuras de datos, teoría de grafos entre otros. Se profundizará el uso de los lenguajes C++ y Java.

Recomendable: Haber llevado o estar llevando cursos de estructuras discretas, también conocer un lenguaje de programación como C++ o java.

Taller de Desarrollo de Aplicaciones para Android Miercoles de 8:30 a 10:45. Laboratorio E, Campus San Lázaro. Julio Santisteban (jsantisteban@ucsp.edu.pe)

El taller proveerá a los participantes con conocimientos de diseño e implementación de aplicaciones para dispositivos Android y sentara la transicion para el desarrollo de aplicaciones para dispositivos iOS, se programara usando el lenguaje de programación Basic para Android, el cual permite desarrollar aplicaciones profesionales en un periodo corto de tiempo.

Recomendable: Alumnos deben haber terminado el 3er semestre o tener un solido conocimiento de programación.


Consideraciones:
  • Se otorgará 1 crédito a quienes acumulen por lo menos 70% de asistencia.
  • Espacios limitados y se considerará estricto orden de inscripción.
  • Cualquier duda sobre fechas de inicio o detalles del taller favor de contactar al respectivo profesor.

La participación es abierta, previo registro.
