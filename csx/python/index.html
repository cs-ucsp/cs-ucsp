Objetivo

Este curso tiene como objetivo introducir al mundo de la programación en Python, aprendiendo de forma gradual, desde conceptos básicos como declaración de variables y funciones a temas más avanzados como programación orientada a objetos.

Dirigido a

Profesores de TIC de colegios secundarios.

Contenido
  • Introducción a Python
  • Ambientes de trabajo
  • Estructuras de control
  • Tipos estructurados: secuencias
  • Funciones
  • Tipos estructurados: clases y diccionarios
  • Ficheros
Metodología
