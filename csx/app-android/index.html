Presentación

Este curso busca capacitar a los profesores de TIC de colegios en creación de aplicaciones para celulares para que puedan enseñar y compartir óptimamente estos conocimientos.

Objetivo
  • Capacitar a los profesores de TIC de colegios en las distintas etapas de creación y programación de aplicaciones para celulares Android.
  • Dar a conocer las tecnologías usadas para la programación de aplicaciones para celulares Android.
  • Desarrollar los componentes necesarios para crear aplicaciones para celulares Android.
  • Conocer los conceptos de creación de aplicaciones para celulares Android. La capacitación se basará en la plataforma AppInventor.
Dirigido a

Profesores de TIC de colegios secundarios

Contenido
  • Introducción a la IDE
  • Introducción a la programación y Estructuras básicas en AppInventor
  • Conceptos de programación para celulares Android
  • Programar, ejecutar y distribuir aplicaciones para celulares Android
